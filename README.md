# odin-recipes
This project is done as part of the theodinproject.com curriculum. For this project, a basic recipe website is built with purely HTML. 
The skills that has been demonstrated with this project being done is:
1. The basic ability of using the linux's terminal to navigate the file manager.
2. The basic ability of using git to clone, add and commit, as well as push to repositories.
3. The ability to code a boilerplate.
4. The ability to make headings on a website with varying sizes from h1 to h6 using HTML.
5. The ability to implement paragraphs on websites using HTML.
6. The ability to implement lists, ordered and unordered on websites with the using HTML.
7. The ability to implement links using anchors which references absolute links or relative links on a website with HTML, to enable navigation between webpages inside the current website, or to enable navigation outside of the website.
8. The ability to implement images which uses absolute links or relative links on websites with HTML.
